## 📌 Descripción

Pipeline completo de procesamiento de imágenes usando OpenCV, que demuestra las principales técnicas de visión por computadora: detección de bordes, contornos, segmentación por color y transformaciones geométricas.

---

## 🎯 Objetivo

Demostrar cómo las máquinas procesan y analizan imágenes aplicando un flujo completo de visión por computadora, desde la imagen original hasta la extracción de características visuales.

## 🔬 Técnicas implementadas

| Paso | Técnica | Descripción |
|---|---|---|
| 1 | Imagen Original | Punto de partida del pipeline |
| 2 | Escala de Grises | Conversión para procesamiento |
| 3 | Detección de Bordes | Algoritmo Canny |
| 4 | Suavizado | Filtro Gaussian Blur |
| 5 | Detección de Contornos | findContours de OpenCV |
| 6 | Segmentación por Color | Máscaras HSV |

---

## 📈 Resultados

El pipeline procesa una imagen de formas geométricas y aplica 6 transformaciones diferentes, demostrando cómo cada técnica extrae información visual distinta de la misma imagen.

---

## 🚀 ¿Cómo ejecutarlo?

1. Abre el archivo `.ipynb` en **Google Colab**
2. Ejecuta las celdas en orden
3. La imagen se genera automáticamente — no hay que descargar nada
4. Cada celda produce una visualización diferente
